"@tenderlove":http://twitter.com/tenderlove/status/5687291469107200 shows us ruby magic to create an infinite hash:

{% highlight ruby %}
>> hash = Hash.new { |h, k| h[k] = Hash.new(&h.default_proc) }
 => {}
>> hash['a'] = 1
 => 1
>> hash['a']
 => 1
>> hash['b']
 => {}
>> hash['b']['c']['d']
 => {}
{% endhighlight %}
